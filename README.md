---

### 🏗️ &nbsp;[Data-Ingress-Framework](https://github.com/Gleybson20/Data-Ingress-Framework)
**Pipeline de Ingestão Modular · Bronze Layer · GCP**

> Framework Python reutilizável para extração e ingestão de dados brutos via APIs REST diretamente no **Google BigQuery** (camada Bronze). Orquestrado via **GitHub Actions** com suporte a múltiplas fontes, versionamento de schema e logging estruturado. Base arquitetural da camada de aterrissagem na **Arquitetura Medalhão**.

`Python` `BigQuery` `GitHub Actions` `REST APIs` `Medallion Architecture`

---

### 🤖 &nbsp;[Signal-Enrichment-Agent](https://github.com/Gleybson20/Signal-Enrichment-Agent)
**AI-Powered Data Enrichment · LLMs at Scale**

> Agente inteligente que utiliza **Large Language Models** para categorização semântica e enriquecimento automático de sinais de dados não estruturados em larga escala. Integra a camada de transformação do pipeline com contexto gerado por IA, aumentando a qualidade e a profundidade analítica dos dados antes da camada Gold.

`Python` `OpenAI API` `LLMs` `NLP` `BigQuery` `Data Quality`

---

### 📡 &nbsp;[API-Tratamento-de-dados-Meta-Ads](https://github.com/Gleybson20/API-Tratamento-de-dados-Meta-Ads)
**Meta Ads API · Data Treatment · Analytics Pipeline**

> Pipeline de extração, tratamento e normalização de dados da **API do Meta Ads** (Facebook/Instagram). Realiza limpeza, tipagem e padronização dos dados de campanhas, adsets e criativos, preparando-os para análise de performance e atribuição. Integrado ao BigQuery para consumo em dashboards de **marketing analytics**.

`Python` `Meta Ads API` `Pandas` `BigQuery` `Marketing Analytics` `ETL`

---

### 🌐 &nbsp;[Algoritmo-de-automação-de-requisições-web](https://github.com/Gleybson20/Algoritmo-de-automa-o-de-requisi-es-web)
**Web Scraping & Request Automation · Data Acquisition**

> Algoritmo de automação de requisições web para coleta sistemática de dados públicos. Implementa estratégias de controle de taxa, retry logic, parsing estruturado e persistência de dados brutos — alimentando a camada de ingestão do data warehouse com fontes não convencionais de dados.

`Python` `Web Automation` `Requests` `Data Acquisition` `Scraping`
